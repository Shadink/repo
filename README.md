### EN
An Animal Crossing Radio which plays songs according to local time and weather.
Installation and execution:
1. In your command line, go to "acmusic" folder
2. Execute command "npm start"
3. A tab will open on your browser and the corresponding song from the Animal Crossing OST will begin playing. The Animal Crossing version (game) can be changed to your preferred one using the buttons situated at the bottom of the video.
4. It's important (but not necessary) to allow autoplay and location access for a better experience.

Made using React.js y and the [OpenWeather API](<https://openweathermap.org/api>). [Animal Crossing: Grass by rush57 on DeviantArt](<https://www.deviantart.com/rush57/art/Animal-Crossing-Grass-145470729>) was used as a background.

### ES
Una radio de canciones de Animal Crossing según la hora y el tiempo locales.
Instalación y ejecución:
1. En tu línea de comandos, entrar en la carpeta "acmusic"
2. Ejecutar el comando "npm start"
3. Se abrirá una página en el navegador y comenzará a sonar la canción correspondiente de la BSO de Animal Crossing. Se puede cambiar de versión (juego) de Animal Crossing al que se prefiera con los botones de debajo del vídeo.
4. Es importante (aunque no imprescindible) permitir autoplay y acceso a la ubicación para una mejor experiencia.

Creado utilizando React.js y la [API de OpenWeather](<https://openweathermap.org/api>). Utilizado como fondo [Animal Crossing: Grass de rush57 en DeviantArt](<https://www.deviantart.com/rush57/art/Animal-Crossing-Grass-145470729>)
